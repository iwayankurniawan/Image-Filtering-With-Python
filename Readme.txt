1. To run this project you need to install Python in you computer first
2. After that you need to install numpy library in your computer as well by write in cmd
pip install numpy 
3. Run the script by write in cmd
cd "Directory of your project"
4. If you already move to that folder, To run the script you can write in cmd
- Run Average Filtering 3X3
	Average_Filtering.py fce5.ppm averageResult3X3.ppm 3
- Run Average Filtering 5X5
	Average_Filtering.py fce5.ppm averageResult3X3.ppm 5
- Run Median Filtering 3x3
	Median_Filtering fce5_salt.ppm medianResult3X3.ppm 3
5. Wait until the process done
